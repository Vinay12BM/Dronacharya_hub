# Dronacharya Hub 📚🚀

**Dronacharya Hub** is a premium, AI-powered educational ecosystem designed to empower students and learners. It integrates advanced machine learning capabilities with localized community resources to create a comprehensive learning environment.

## ✨ Features

- **Tutor Module**:
  - Interactive course dashboards and video lessons.
  - AI-powered quiz generation based on course topics (using Gemini API).
  - Real-time learning assistance with AI chat support.
  - Text-to-Speech (TTS) integration for accessible learning.
- **Research Module**:
  - AI-assisted generation of scholarly research papers.
  - Automatic citation generation (APA, MLA, etc.).
  - Document export functionality (DOCS, PDF).
- **Book Marketplace**:
  - Geolocation-based book search to find resources "nearby".
  - Buy, swap, or donate textbooks and educational materials.
  - WhatsApp integration for direct communication with sellers.
- **Modern UI**:
  - Stunning, premium design with glassmorphism and smooth animations.
  - Fully responsive for all devices.

## 🛠️ Technology Stack

- **Backend**: Flask (Python)
- **Database**: PostgreSQL (via SQLAlchemy)
- **AI/ML**: Gemini AI API
- **Frontend**: HTML5, Vanilla CSS (Premium styling), JavaScript
- **Deployment**: Render / Gunicorn

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Suprith-3/Drrona.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your `.env` file with your Gemini API key and database URL.
4. Run the application:
   ```bash
   python run.py
   ```
